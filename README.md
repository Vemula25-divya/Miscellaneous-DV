# Drug Discovery Utilities

A collection of supporting notebooks, utility scripts, exploratory analyses, and workflow automation tools developed during computational drug discovery research.

## Overview

This repository contains miscellaneous notebooks that support various stages of computational drug discovery, cheminformatics, molecular modeling, data analysis, and research workflow development.

## Contents

The repository includes notebooks related to:

* Data processing and curation
* Cheminformatics analyses
* Research workflow automation
* Exploratory data analysis
* Computational biology applications
* Supporting research utilities

## Applications

* Antimicrobial drug discovery
* Structure-based drug design
* Virtual screening support
* Molecular modeling
* Data analysis and visualization

## Technologies

* Python
* Pandas
* NumPy
* RDKit
* Scikit-learn
* Jupyter Notebook
